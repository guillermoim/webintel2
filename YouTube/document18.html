    Habilitar JavaScript

    Debes habilitar JavaScript para poder utilizar varias funciones. Para ello, sigue las instrucciones que se indican a continuación.

    1. Si aún no lo has hecho, accede a la aplicación del navegador en el dispositivo.
    2. Toca el botón Menú de tu dispositivo.  
    3. Selecciona Más.
    4. Selecciona Ajustes.
    5. Marca la casilla situada al lado de "Habilitar JavaScript".
    6. Sal y vuelve a cargar el sitio web al que intentabas acceder en el navegador.

    Otros

    Te recomendamos que consultes el sitio web de ayuda online o el manual de usuario del dispositivo para obtener más instrucciones sobre cómo habilitar JavaScript en el navegador web.
